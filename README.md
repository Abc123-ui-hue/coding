# VANTAGE - Premium Footwear Store

A modern e-commerce platform for high-performance shoes and accessories.

## Features

- **Storefront**: Browse a curated selection of premium footwear.
- **Product Filtering**: Sort and filter by category (Men, Women, Kids) and color.
- **Cart & Checkout**: Full shopping cart functionality with a secure checkout process.
- **Wishlist**: Save your favorite pairs for later.
- **Admin Dashboard**: Real-time sales data and inventory management for store owners.
- **Modern UI**: Clean, responsive layout optimized for all devices.

## Tech Stack

- **Frontend**: React.js
- **Tooling**: Vite
- **Styling**: Custom CSS
- **State**: React Context API

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run development server:
   ```bash
   npm run dev
   ```

## Project Structure

- `src/components`: UI components and sections.
- `src/pages`: Main application pages.
- `src/context`: Global state and business logic.
- `src/admin`: Dashboard and inventory tools.
- `src/assets`: Static styles and images.
